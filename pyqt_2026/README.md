6. Slozitejsi_Widgety_2026.ipynb

8. Obrazky_a_Stylovani_2026.ipynb

## Poznámky
- PyQt aplikace spouštějte lokálně (ne v Google Colab).
- Instalace knihovny: `pip install PyQt5`
- Původní notebooky zůstaly beze změny ve složce `PyQt`.
