# PyQt6 2026

Materiály k PyQt6 ve stylu SQL notebooků (stručné vysvětlení + ukázky + cvičení + očekávaný výstup).

## Doporučené pořadí

| # | Soubor | Téma |
|---|---|---|
| 01 | `01 - Okno_tlacitko_napis_2026.ipynb` | První okno, QLabel, QPushButton |
| 02 | `02 - Funkce_u_tlacitka_2026.ipynb` | Signály a sloty, události tlačítek |
| 03 | `03 - Uzivatelsky_vstup_2026.ipynb` | QLineEdit, QSpinBox, vstup od uživatele |
| 04 | `04 - LayoutManager_2026.ipynb` | QVBoxLayout, QHBoxLayout, QGridLayout |
| 05 | `05 - Viceokenni_aplikace_2026.ipynb` | Více oken, přepínání, modální dialogy |
| 06 | `06 - Slozitejsi_Widgety_2026.ipynb` | QCheckBox, QRadioButton, QComboBox, QSlider |
| 07 | `07 - Dialogy_a_Upozorneni_2026.ipynb` | QMessageBox, QFileDialog, QInputDialog |
| 08 | `08 - Obrazky_a_Stylovani_2026.ipynb` | QPixmap, QSS styly, vlastní vzhled |
| 09 | `09 - OOP_QMainWindow_2026.ipynb` | QMainWindow, OOP struktura aplikace |
| 10 | `10 - Menu_Toolbar_StatusBar_2026.ipynb` | QMenuBar, QToolBar, QStatusBar, QAction |
| 11 | `11 - QTableWidget_2026.ipynb` | QTableWidget, zobrazení dat v tabulce |
| 12 | `12 - Timery_animace_2026.ipynb` | QTimer, animace, opakované akce |

## Poznámky
- PyQt6 aplikace spouštějte **lokálně** (ne v Google Colab).
- Instalace knihovny: `pip install PyQt6`
- `QAction` je v PyQt6 součástí `PyQt6.QtGui` (ne `QtWidgets` jako v PyQt5).
